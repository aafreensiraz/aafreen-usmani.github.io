1 How to get your Instagram data
  1. Open Instagram → Profile → Menu (☰) → Settings → Your Activity → Download Your Information
  2. Select JSON format (preferred) or HTML format — both work!
  3. Request download and wait for the email from Instagram
  4. Extract the ZIP file — you'll find folders like content/, messages/ with .json or .html files
  5. Upload the JSON files below ↓
